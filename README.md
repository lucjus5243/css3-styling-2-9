# Deployment and Local Hosting – Project 5.8

## Project Overview
This project is a responsive website created using HTML5, CSS3, and JavaScript. The purpose of this assignment is to demonstrate real-world deployment skills by documenting the project, publishing the site using GitHub Pages, and running the website locally using XAMPP.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages
- XAMPP (Apache)

## GitHub Pages Live Site

- `styles-contact.css` – Contact page styling
- `script.js` – JavaScript functionality
